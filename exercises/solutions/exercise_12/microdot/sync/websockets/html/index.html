WEB sockets for picoweb

Sending SHT30 sensor data over websockets with picoweb

This is a WEB server, based on picoweb, which sends SHT30 temperature and humidity data to a browser using websockets. Also via websockets it receives information in which state the user LED on the ESP32 should be and it controls the LED correspondingly. Please try IP_of_WebServer/sensor.html to see the result.
The program has been written for the Course on the Internet of Things (IoT) at the University of Cape Coast (Ghana)
